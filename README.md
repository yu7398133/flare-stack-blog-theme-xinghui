# Flare Stack Blog Theme: Xinghui

Glassmorphism风格主题，灵感来源于 [XinghuisamaBlogs](https://github.com/heiehiehi/XinghuisamaBlogs)。

## 特性

- 🎨 Glassmorphism毛玻璃风格设计
- 🎵 内置音乐播放器（网易云音乐API）
- 📸 照片墙、说说、项目展示
- 🌙 暗色/亮色主题切换
- 📱 完全响应式设计
- ✨ 弹幕背景、入场动画

## 使用方法

在flare-stack-blog项目中设置环境变量：
```
THEME=xinghui
```
